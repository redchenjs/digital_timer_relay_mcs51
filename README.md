Digital Timer Relay
===================

Digital Timer Relay based on STC89C51RC chip.

## Dependencies

```
sdcc
stcgal
```

## Preparing

### Obtain the source

```
git clone --recursive https://github.com/redchenjs/digital_timer_relay_mcs51.git
```

### Update an existing repository

```
git pull
git submodule update --init --recursive
```

## Building

```
make flash
```
